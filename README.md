# Fraud Detection with Deep Learning

This repository contains a Fraud Detection system built using Deep Learning techniques. The model is trained on a large-scale dataset to ensure robustness and accuracy in identifying fraudulent transactions.

## Key Features

- **Deep Learning Model**: Utilizes state-of-the-art Deep Learning techniques for fraud detection.
- **Dataset**: Trained on a comprehensive dataset to enhance performance.
- **Metrics**:
  - **Training Metrics**:
    - Accuracy: 0.99611
    - Loss: 0.01493
  - **Validation Metrics**:
    - Accuracy: 0.99918
    - Loss: 0.00366
- **Accuracy Score**: Achieved 0.99897 using `accuracy_score(y_test, y_pred)`.
